# Isamani Tours — Flask Web Application

> Fork this repo, work on it, then open a Pull Request with your changes.

---

## Setup Instructions

```bash
git clone https://github.com/isamani-tours/isamani-flask-app.git
cd isamani-flask-app
python -m venv venv
source venv/bin/activate         # Linux/Mac
venv\Scripts\activate            # Windows
pip install -r requirements.txt
cp .env.example .env
python run.py
```

Visit `http://127.0.0.1:5000`

---

## About the Project

This is a tour company web application built with Flask and SQLite. It includes:

- User authentication (register, login, logout)
- Blog (create, edit, delete posts)
- Itinerary builder (plan day-by-day tour schedules)
- Chatbot assistant

## Contribution Rules

1. **Fork** the repo — do NOT push directly to main
2. Create a branch: `git checkout -b your-name`
3. Write clear commit messages
4. Open a **Pull Request** when done — describe what you changed and why
5. Do not break existing features when adding new ones
